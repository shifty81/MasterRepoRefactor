# Titan and Seasonal Loop

The endgame is a seasonal race to build a Titan and escape a collapsing universe.

## Seasonal loop

```text
Spawn
 -> progress
 -> expand
 -> compete
 -> build Titan
 -> activate gate
 -> universe collapse
 -> reset season
```

## Titan core parts

- Titan Core
- Frame
- Power Systems
- Stabilizers
- Navigation Matrix
- Gate Engine
