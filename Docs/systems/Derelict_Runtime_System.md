# Derelict Runtime System

Derelicts are state transformations of ships.

## Transformation path

```text
Operational Ship
 -> Damaged
 -> Disabled
 -> Derelict
 -> Cleared
```

## Core features

- breach entry
- offline/partial power states
- interior modification
- salvage-dense targets
- persistent cleared state later

## Entry methods

- repaired airlock
- breach entry (primary in MVP)
