# Equipment and Stat Modifier System

Gear is modular, slot-driven, visual, and stat-bearing.

## Core slots

- rig helmet
- rig torso
- rig gloves
- rig boots
- rig pack
- primary tool

## Core stats

- oxygen efficiency
- EVA stability
- salvage yield bonus
- repair efficiency
- cutting efficiency
- inventory capacity
- mech interface compatibility
- ship system efficiency

## Core rule

All meaningful loot should matter visually or mechanically.
