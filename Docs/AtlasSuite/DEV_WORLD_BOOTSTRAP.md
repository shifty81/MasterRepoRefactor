# Dev World Bootstrap

## Goal
One internal world used to test all major systems from inside Atlas Suite.

## Required zones
- Spawn room
- Rig terminal
- Inventory/crafting terminal
- Airlock and pressure room
- EVA lane
- Mech bay
- Ship hangar
- Builder pad
- Combat range
- Salvage lane
- Mission board
- Economy debug terminal
- Faction debug terminal
- Save/load checkpoint

## Test flow
```text
Spawn Rig
→ Equip starter tools
→ Open airlock
→ EVA test
→ Enter mech
→ Exit mech
→ Enter ship
→ Place builder part
→ Damage target
→ Salvage wreck part
→ Store item
→ Accept mission
→ Save
→ Reload
```
