# Project Vision

Master Repo is a voxel + low-poly sci-fi sandbox/survival/salvage game framework.

## Identity

- Voxels define structure, damage, mining, repair, and procedural generation.
- Low-poly assets define style, readability, and modular gameplay presentation.
- The player uses a **rig**: the in-game term for the combined space suit and backpack assembly.
- Progression is tiered across all equippable systems.
- The world is seasonal, configurable, and usually server-authoritative.

## Core fantasy

A player equips a rig, operates ships and mechs, performs EVA salvage, explores derelicts, upgrades gear and vehicles, completes faction contracts, and works toward constructing a Titan to escape a collapsing universe.

## Core principles

1. Everything modular.
2. Everything schema-driven.
3. Everything tier-aware.
4. Everything validated.
5. Existing code that does not conform gets refactored.

## Style rules

- Readability first
- Chunk-based construction
- Functional visual language
- Low poly, not low detail
- Shared material logic across rigs, mechs, ships, interiors, and props
