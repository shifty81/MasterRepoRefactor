# Scripts

This folder contains automation scripts for setup, build, validation, and CI.

## Subfolders

| Folder | Purpose |
|--------|---------|
| `Bootstrap/` | First-time environment setup scripts |
| `Build/` | Build helper scripts |
| `Validate/` | Code and data validation scripts |
| `CI/` | CI/CD helper scripts |

## Usage

Scripts should be run from the repository root unless otherwise noted.
